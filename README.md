# Muhammad Ahmad Humayoun Portfolio

A stunning neon-themed portfolio showcasing immersive technology expertise.

## Features

- Modern neon design with cyan/magenta theme
- 3D animated elements and particle effects
- Custom cursor with trail effect
- Scroll animations and parallax effects
- Fully responsive design

## Quick Start

### Local Development

```bash
# Clone the repo
git clone https://github.com/yourusername/mah-portfolio.git
cd mah-portfolio

# Open in browser
open index.html
```

### Deploy on GitHub Pages

1. Push this repository to GitHub
2. Go to Settings > Pages
3. Select "Deploy from a branch" > main
4. Your site will be live at `https://yourusername.github.io/repo-name`

## Customization

Edit `index.html` to update:
- Personal information
- Projects and experience
- Skills and expertise
- Social links

## Tech Stack

- HTML5
- CSS3 (Animations, Flexbox, Grid)
- Vanilla JavaScript
- Google Fonts (Syne, Inter, Space Grotesk)
